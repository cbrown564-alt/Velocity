# Documentation Archive

Archived documents are historical evidence, not current authority.

Use active docs in `docs/README.md` for strategy, architecture, design, and execution. If an archived document contains a still-valid constraint, extract that constraint into an active architecture, design, tracker, or decision document instead of linking to the archived file as live guidance.
