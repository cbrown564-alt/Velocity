# Report Quality Exemplars

The v2 plan requires a north-star brand-tracker exemplar before generated decks are scored as "good."

Current state:

- `brand_tracker_north_star_candidate.md`: local candidate story frame and slide contract.
- `brand_tracker_north_star_signoff.md`: required human consultant sign-off checklist.

This folder intentionally distinguishes a Codex-generated candidate from a signed-off exemplar. The candidate can be used to exercise `exemplar_diff.md`; it should not be used as external quality proof until the sign-off checklist is complete.
